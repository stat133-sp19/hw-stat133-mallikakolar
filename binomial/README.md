## Overview

This is a binomial package. 

It contains functions relating to binomial random variables.
There are 3 types of functions: private auxiliary summary functions, private checker functions, and main functions.

This package contains the following functions:  

**Summary Functions**
`aux_mean()`
`aux_variance()`
`aux_mode()`
`aux_skewness()`
`aux_kurtosis()`

**Checker Functions**
`check_prob()`
`check_trials()`
`check_success()`

**Main Functions**
`bin_choose()`
`bin_probability()`
`bin_distribution()`
`bin_cumulative()`
`bin_variable()`
`bin_mean()`
`bin_variance()`
`bin_mode()`
`bin_skewness()`
`bin_kurtosis()`
